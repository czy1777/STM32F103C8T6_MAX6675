# STM32F103C8T6_MAX6675
STM32F103C8T6 HAL 库驱动 MAX6675 测温模块

最近在项目开发中，我需要通过 MAX6675 模块获取温度。但查阅资料时发现，目前基于 HAL 库的开发资料很少，大多是标准库实现。由于我的代码框架已经采用 HAL 库搭建，为这一个模块切换到标准库会非常繁琐。因此，我通过查阅各大技术网站并结合 AI 咨询，最终完成了基于 STM32F103C8T6 的 HAL 库驱动 MAX6675 模块的实现，特此记录分享。

本文的核心重点，将放在该模块的实际使用方法讲解上。因为 MAX6675 模块的原理相关内容在网上已有大量资料和博客可供参考。
